食保宝临期食品网站设计  

作者：李恒鑫、王家梁、饶稷、李松泽

# 项目开发环境

* Windows 系统
* 后端：IDEA、Springboot、mysql、express
* 前端：Visual Studio Code、Vue-cli3、node.js

# 如何运行本项目

* 启动springboot后端
* 进入 AlipayServe 文件夹，打开命令提示符（cmd窗口），输入`npm install`、`node app.js` 启动支付宝组件
* 在命令提示符中进入 frontend 文件夹，输入`npm install`、 `npm run serve` 启动前端代码。
* 至此，项目已经可以正常使用。
